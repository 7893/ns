# 开发指南

## 开发环境设置

### 本地环境
- Python 3.13虚拟环境
- 必要依赖包安装
- 环境变量配置
- 开发工具配置

### 代码规范
- 代码签名必须启用
- commit消息使用英文，不超过10个单词
- 遵循Python PEP 8编码标准
- 函数和类的文档字符串要求

## 项目结构

### 核心目录
- `apps/`: 各个数据收集器应用
- `packages/`: 共享库和工具包
- `infra/`: 基础设施配置文件
- `scripts/`: 部署和维护脚本

### 配置文件
- `.env`: 环境变量配置
- `requirements.txt`: Python依赖
- `.gitignore`: Git忽略规则
- `.tool-versions`: 工具版本管理
