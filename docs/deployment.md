# 部署指南

## 环境要求

### 基础环境
- Google Cloud Platform账号（已启用计费）
- 项目已启用必要的API
- 服务账号具备适当权限
- NASA API密钥

### 必需工具
- Python 3.13+
- Terraform 1.13.3
- Google Cloud CLI

## 部署方式

### GitHub Actions自动部署（推荐）
- 代码推送到main分支自动触发
- 包含基础设施配置
- 自动处理服务配置
- 提供部署状态反馈

配置GitHub Secrets：
- `GCP_SA_KEY`: GCP服务账号JSON密钥
- `NASA_API_KEY`: NASA API密钥

### 本地手动部署
用于开发和测试：
- 需要本地环境配置
- 手动基础设施配置
- 逐步服务部署
- 手动配置验证
